DISCOVERY FOLDER REDESIGN

Music is magic!

This project was about restyling the kids' catalog of Orchestra Sinfonica di Milano containing all the events for this target. The important part of a redesign is that it needs to have a purpose.

For this project we didn't just retouch the single pages, we created illustrations in order to make the book more appealing for the young audiences and we created a coherent color and typography system which was sometimes missing in the original folder.
